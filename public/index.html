About LiteracyMap

LiteracyMap is a reading intervention tracker designed for classroom teachers and reading specialists. It supports MTSS/RTI frameworks by providing:

  • Tier-based student roster management
  • Multi-measure assessment tracking (DRA, F&P, WPM, comprehension, phonics)
  • Qualitative observation logging
  • Intervention program planning and progress monitoring
  • Group builder for small-group instruction
  • Evidence-based strategy library
  • MTSS report generation
